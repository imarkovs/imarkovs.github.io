Software

The computational algorithms described in my publications are implemented and the software is available together with the pdf files of the papers. This makes the reported simulation results reproducible. Since 2008, I've been using the literate programming style and the noweb system for literate programming. The perfect reproducible research solution for me is org-babel, which combines literate programming with noweb and code evaluation within the document, see this presentation.
